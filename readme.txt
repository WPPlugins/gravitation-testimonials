A Plugin to integrate testimonials

== Description ==
Gravitation Testimonials, is a plugin to display testimonials on your site, with ShortCodes

== Installation ==
Download and install from WordPress.org.
Configure as you see fit, under Gravitation / Gravitation Testimonials.
1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place [gravitaiton_testimonials] in your templates
